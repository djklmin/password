# 更新日志

## [1.2.0] - 2026-03-07

### 新增
- ✏️ 编辑密码功能 - 支持编辑所有类型的密码项目（登录凭证、安全笔记、银行卡、身份信息）
- 🔐 登录失败限制 - 5次失败后锁定15分钟，防止暴力破解

### 修复
- 修复密码验证逻辑 - 确保错误密码无法通过验证

### 改进
- 优化编辑模态框 UI，支持密码生成器和强度检测
- 登录页面显示剩余尝试次数警告
- 锁定期间显示倒计时

---

## [1.1.0] - 2026-03-06

### 新增
- 🔄 初始化系统 - 首次使用自动创建管理员账户
- ⚙️ 设置页面 - 修改密码、CSV导入、重置数据库
- 🎨 个性化设置 - 自定义网站标题和图标
- 👤 用户名登录 - 改用用户名替代邮箱登录
- ⏰ 持久会话 - 延长登录状态有效期
- 📄 版权信息 - 页脚添加版权声明

### 改进
- 登录页面简化，移除注册选项
- 中文界面优化

---

## [1.0.0] - 2026-03-05

### 新增
- 🔒 端到端加密 - AES-256-CBC + PBKDF2 密钥派生
- 🔑 零知识架构 - 服务器无法访问明文数据
- 🛡️ 二次验证 (2FA) - TOTP 时间基验证码
- 📦 多类型支持 - 登录凭证、安全笔记、银行卡、身份信息
- 📁 文件夹分类 - 自定义组织管理
- ⭐ 收藏功能 - 快速访问常用项目
- 🔧 密码生成器 - 可自定义长度和字符类型
- 📤 数据导出 - JSON 格式备份
- 🚀 Vercel 部署支持
